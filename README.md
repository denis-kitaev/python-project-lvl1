## Описание
Набор консольных игр. Игроку необходимо ответить на три вопроса подряд. Ответив правильно, игрок побеждает. Ошибка в ответе завершает игру.

## Список игр
* brain-even
* brain-calc
* brain-gcd
* brain-progression
* brain-prime

## Установка
```
make build
make package-install
```
